## Como usar

1. **Bateu dúvida** num tema? Abre o [mapa](mapa.md), acha o tema e relê o arquivo.
2. **Estude na ordem das dependências** — cada tema diz o que precisa vir antes. Se a 2ª condicional está confusa, o problema costuma estar na 1ª condicional ou no passado simples.
3. Cada lição termina com **Revisão rápida (5 min)** — perfeita pros intervalos do dia.
